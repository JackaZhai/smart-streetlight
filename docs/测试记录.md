# 测试记录

## 计划验证项

| 项目 | 命令 | 目标 |
| --- | --- | --- |
| 后端领域测试 | `npm --workspace backend test` | 验证遥测入库、自动控制、离线告警和阈值统计 |
| 全量测试 | `npm test` | 运行所有 workspace 测试 |
| 类型检查 | `npm run typecheck` | 验证 TypeScript 类型 |
| 构建 | `npm run build` | 验证前端、后端、mock-device 可编译 |

## 人工联调流程

1. `docker compose -f deploy/docker-compose.yml up -d`
2. `npm run dev:backend`
3. `npm run dev:frontend`
4. `npm run dev:mock`
5. 打开 `http://localhost:5173`
6. 检查实时光照、历史趋势、设备表格、告警列表和智能问答。
7. 点击“开灯/关灯”，确认 mock-device 输出执行日志。
